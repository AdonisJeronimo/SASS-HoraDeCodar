Partials

Os partials são uma forma de dividir o código CSS em arquivos menores e mais organizados. Ela envolve o uso de arquivos parciais (partials) que contêm trechos de código CSS ou SASS reutilizáveis e, em seguida, são importados em um arquivo principal para compilação.

Para criar um arquivo parcial, você precisa nomeá-lo com um prefixo de _ (underline) no início do nome do arquivo, por exemplo _variables.scss. Essa convenção de nomenclatura é uma convenção comum para identificar arquivos que não são os principais do projeto, sendo ele um partials que irá ter uma inclusão em outro arquivo. Ao importar um arquivo parcial em um arquivo principal do SASS, você usa a diretiva @import. Aqui está um exemplo:

@import 'variables'

Mixins

É uma funcionalidade que nos permite agrupar um conjunto de propriedades CSS em um bloco de código reutilizável. Com mixins, você pode definir estilos complexos ou repetitivos uma vez e usá-los em vários lugares do seu código. Para criar um mixin no SASS, você usa a diretiva @mixin seguida de um nome para o mixin e um bloco de código contendo as propriedades CSS desejadas. Aqui está um exemplo simples:

@mixin efeito-hover propriedade

Mixins com parâmetros

Os mixins no SASS também podem receber parâmetros, permitindo torná-los ainda mais flexíveis e dinâmicos. Os parâmetros são definidos entre parênteses ao declarar o mixin e podem ser usados dentro do bloco de código do mixin. Caso passemos mais de um parâmetro, separamo-los por vírgulas. EXEMPLO:
