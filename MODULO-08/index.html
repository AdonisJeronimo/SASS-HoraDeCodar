Módulo 7

Módulo COLORS

O módulo de cores no SASS deve ser importado com o @use ‘sass:list’ e ele fornece várias funcionalidades para trabalhar com cores de forma mais conveniente. Ele inclui uma variedade de funções embutidas que permitem criar, manipular e combinar cores de maneira flexível. Como por exemplo:

  • color.adjust: Aumenta ou diminui alguma propriedade da cor.
  • color.green: Retorna o quanto de verde tem uma cor.
  • color.scale: Pode deixar uma cor mais clara ou escura.
  • color.invert: Interte uma cor.

Módulo COLORS II

Algumas das manipulações mais usadas.

  • lighten($color, $amount): Clareia a cor especificada pelo valor indicado. O parâmetro $amount representa a porcentagem de clareamento.
  • darken($color, $amount): Escurece a cor especificada pelo valor indicado. O parâmetro $amount representa a porcentagem de escurecimento.
  • saturate($color, $amount): Aumenta a saturação da cor especificada pelo valor indicado. O parâmetro $amount representa a porcentagem de aumento da saturação.
  • desaturate($color, $amount): Diminui a saturação da cor especificada pelo valor indicado. O parâmetro $amount representa a porcentagem de diminuição da saturação.
  • adjust-hue($color, $degrees): Altera o matiz da cor especificada em um determinado ângulo. O parâmetro $degrees representa o ângulo de rotação do matiz.
  • mix($color1, $color2, $weight): Combina duas cores especificadas, criando uma nova cor. O parâmetro $weight determina a proporção das duas cores na mistura.

Módulo LIST

O módulo de list é importado utilizando o **@use ‘sass:list’** e ele oferece funcionalidades para trabalhar com listas de valores. Ele inclui várias funções embutidas que permitem manipular, acessar e transformar listas de forma conveniente.Algumas das principais funcionalidades do módulo de listas no SASS são:

  • length($list): Retorna o comprimento (número de itens) de uma lista.
  • nth($list, $n): Retorna o valor na posição especificada $n em uma lista.
  • index($list, $value): Retorna a posição do valor especificado $value em uma lista
  • join($list1, $list2, $separator): Combina duas listas em uma única lista. O parâmetro opcional $separator especifica o separador a ser usado entre os itens.
  • append($list, $value, $separator): Adiciona um valor ao final de uma lista. O parâmetro opcional $separator especifica o separador a ser usado entre os itens.
  • zip($lists...): Combina várias listas em uma única lista, agrupando os valores correspondentes de cada lista em uma sublista.

Essas são apenas algumas das funcionalidades oferecidas pelo módulo de listas no SASS. Ele permite a manipulação eficiente de conjuntos de valores relacionados, como cores, tamanhos, propriedades CSS, entre outros. Com as funções do módulo de listas, você pode acessar itens específicos, combinar listas, adicionar valores e realizar várias operações em listas no seu código SASS.

Módulo MAP

  • Criação de mapas: Você pode criar mapas usando a função map() e especificando pares chave-valor. Por exemplo: $map: (chave1: valor1, chave2: valor2, chave3: valor3).
  • Acesso aos valores: É possível acessar os valores em um mapa usando a função map-get(), passando o nome do mapa e a chave desejada. Por exemplo: map-get($map, chave1) retorna o valor associado à chave1 no mapa.
  • Atualização de valores: Você pode atualizar um valor existente em um mapa usando a função map-merge(), que combina dois mapas. Por exemplo: $mapAtualizado: map-merge($mapExistente, (chave1: novoValor)).
  • Verificação de existência de chave: A função map-has-key() verifica se uma chave específica está presente em um mapa. Ela retorna true se a chave existir e false caso contrário.
  • Iteração em mapas: O SASS permite percorrer um mapa usando a diretiva @each. Você pode iterar sobre as chaves e valores do mapa e realizar operações em cada par.

Módulo MATH

O módulo Math no SASS fornece uma série de funções matemáticas embutidas que permitem realizar cálculos e operações matemáticas em estilos. Ele permite que você execute operações matemáticas diretamente no seu código SASS, facilitando o cálculo de valores, tamanhos e propriedades relacionadas.Algumas das funções matemáticas disponíveis no módulo Math do SASS incluem:

  • abs($number): Retorna o valor absoluto de um número.
  • ceil($number): Arredonda um número para cima para o próximo número inteiro.
  • floor($number): Arredonda um número para baixo para o número inteiro anterior.
  • round($number): Arredonda um número para o número inteiro mais próximo.
  • min($numbers...): Retorna o valor mínimo entre vários números.
  • max($numbers...): Retorna o valor máximo entre vários números.
  • random(): Gera um número aleatório entre 0 e 1.

Essas são apenas algumas das funções matemáticas disponíveis no módulo Math do SASS. Elas permitem que você realize operações matemáticas avançadas, como arredondamento, cálculos de valor absoluto, mínimo e máximo, entre outros. O uso dessas funções no seu código SASS torna mais fácil e eficiente lidar com cálculos e manipulação de valores numéricos.

Módulo META

Quando você usa **`@use 'sass:meta'`**, está importando o namespace **`meta`** do SASS. O namespace **`meta`** contém variáveis e funções internas do SASS que podem ser usadas para acessar informações e metadados relacionados ao ambiente do SASS.Algumas das funcionalidades que podem ser acessadas através do namespace **`meta`** são:

  • meta.module-variables($module): Retorna uma lista de variáveis definidas em um módulo específico.
  • meta.module-functions($module): Retorna uma lista de funções definidas em um módulo específico.
  • meta.inspect($value): Retorna uma representação em string de um valor, útil para depuração e visualização de informações.

Essas são apenas algumas das funcionalidades disponíveis através do namespace meta. Elas permitem acessar e explorar informações sobre os módulos e elementos do SASS, como variáveis e funções, durante o processo de compilação.

No entanto, é importante observar que o uso do namespace meta é geralmente avançado e não é recomendado para o uso comum no desenvolvimento de estilos. É mais útil em casos específicos em que você precisa acessar informações internas do SASS ou realizar tarefas mais avançadas relacionadas à manipulação do ambiente do SASS.

Módulo STRING

O módulo de strings do SASS fornece funcionalidades para manipulação de strings, permitindo realizar operações e transformações em valores de texto. Com o módulo de strings, você pode realizar tarefas como concatenação, formatação, extração de partes e manipulação de strings de forma conveniente.Algumas das funcionalidades disponíveis no módulo de strings do SASS são:

  • str-length($string): Retorna o comprimento de uma string, ou seja, o número de caracteres que ela contém.
  • str-insert($string, $insert, $index): Insere uma string $insert dentro de outra string $string em uma posição específica $index.
  • str-index($string, $substring): Retorna a posição da primeira ocorrência de $substring dentro de $string.
  • str-slice($string, $start, $end): Extrai uma parte específica de uma string, começando do índice $start até o índice $end (opcional).
  • to-upper-case($string): Converte uma string para letras maiúsculas.
  • to-lower-case($string): Converte uma string para letras minúsculas.
  • str-join($strings, $separator): Combina várias strings em uma única string, separadas por um $separator (opcional).

Essas são apenas algumas das funcionalidades oferecidas pelo módulo de strings no SASS. Elas permitem que você manipule e transforme valores de texto de forma flexível e eficiente dentro do seu código SASS.
